治理闭环示意

业务调用 → execute_cypher / bulk_insert
   ↓ (audit 自动写 AuditAction + AuditRef)
   ↓ (异常检测：0 行 / 抛错 / 边约束不符)
   ↓
flag_for_review → 创建 PendingReview + TRIGGERED 边
   ↓
审核员 (本页 3 按钮)：
   ├─ approve  → 接受原结果（仅写 HumanDecision）
   ├─ reject   → 拒绝（状态置为 rejected）
   └─ override → 用修正数据走 bulk_insert 的 MERGE 灌入（写 audit: manual_commit）
